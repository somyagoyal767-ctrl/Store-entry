# IOS-APPLE-STORE-PROJECT
PROJECT TITLE: 

IOS APPLE STORE

OVERVIEW OF PROJECT:

The IOS APPLE STORE is a Python-based interactive online platform for shopping APPLE product. 
The application starts with sign up and login for both new and existing users.
Then it prints the lists of products avialable along with price and asks user to selects the products they want.
lastly program asks user to enter their delievery address and prints a detailed bill

FEATURES:

An online shooping platform
Unique sign up and login features for both new and existing users
Shows a the lists of products and price in a table form
User can easily selects products they want along with quantity
Required address for quick delievery
Prints a detailed bill for the customer

TECHNOLOGIES USED:

Programming Language: Python 3.x
Libraries Used: tabulate(organising data in tabular form), date and time (print present date and time )
IDE/Environment: Any Python IDE or text editor (e.g. Google Colab)
